# Trabajo 1 UNED – Asociación de Comerciantes de Haro

## Objetivo
Portal web maquetado con HTML5 y CSS3 siguiendo los requisitos del enunciado.

## Estructura del proyecto
- index.html
- style.css
- img/ (recursos gráficos)
- (pendiente) inscripcion.html (formulario HTML5)
- (opcional) noticias.html (ampliación)

## Cobertura de requisitos (enunciado)
- Cabecera y pie (HTML5): OK
- Menú de navegación: OK
- Zona de noticias: OK (section + article + time datetime)
- Imágenes y datos verosímiles: OK
- CSS3 bordes curvos y sombras: OK
- Formulario inscripción HTML5: PENDIENTE (se implementará en inscripcion.html)
- Uso de “capas HTML5” (estructura semántica): OK

## Decisiones de diseño (didácticas)
- HTML5 semántico: header/nav/main/section/article/figure/footer
- Layout principal con CSS Grid
- Componentes repetibles (cards) para comercios

## Control de versiones
- Rama main: versión estable
- Rama optimizacion_didactica: limpieza, cobertura y documentación
